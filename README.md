# Used technologies

- Angular used as frontend framework

- Styles and many components take from Angular Material

# How to run

## With node

For prod version:

From root
- ```npm i```
- ```npm run build```
- ```npm start```

For dev version:

From root
- ```npm i```
- ```npm run start:watch```
